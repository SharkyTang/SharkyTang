# GitHub Profile README Setup

1. Create a public GitHub repository named exactly `SharkyTang`.
2. Put `README.md` from this folder into that repository.
3. Copy the `.github/workflows/snake.yml` file into the same repository.
4. Commit and push.
5. In the repository, open the `Actions` tab and run `Generate contribution snake` once.
6. Go to `https://github.com/SharkyTang` and refresh the profile page.

The snake image will show after the workflow creates the `output` branch. If it does not appear immediately, wait a minute and refresh.
